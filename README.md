# HunarHub – Digital Marketplace for Local Micro-Entrepreneurs

## 🚀 Project Overview

HunarHub is a platform that connects local micro-entrepreneurs with customers, enabling them to showcase and sell their products online.

## 💡 Features

* User Registration & Login
* Product Listings
* Search & Filter Products
* Dashboard for sellers
* Responsive UI

## 🛠 Tech Stack

* Frontend: React
* Backend: Django
* Database: SQLite / PostgreSQL

## ▶️ How to Run

1. Clone the repo
2. Install dependencies
3. Run frontend and backend
